# pragma-knowledge

Repositorio de conocimiento global de Pragma. Fuente única de verdad para el conocimiento técnico, metodológico y de negocio de la organización.

## Propósito

Alinear a todos los colaboradores bajo un mismo marco de trabajo y servir como el **Grafo de Conocimiento Activo (Knowledge Base)** que alimenta a los Agentes de IA de Pragma.

## Estructura

```
pragma-knowledge/
├── manifest.json               # Índice y metadatos dinámicos
├── business-domain/            # Conocimiento por industria
├── chapters/                   # Conocimiento por especialidad técnica
├── organizational-rules/       # Marco normativo corporativo global
└── meta/                       # Gobierno del estándar
```

## Cómo contribuir

1. Leer el estándar completo: `standard_sopp.md`
2. Crear rama desde trunk: `feature/[descripcion-corta]`
3. Seguir las plantillas oficiales en `meta/schema.md`
4. Abrir Merge Request y esperar revisión

## Reglas de Oro

1. Nomenclatura en inglés y kebab-case
2. Contenido en español técnico
3. No duplicidad (DRY)
4. Contexto explícito (AI-Ready)
5. Atomicidad — un archivo, un propósito
